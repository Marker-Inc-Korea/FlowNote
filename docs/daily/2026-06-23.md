# 2026-06-23 작업 기록

이 문서는 과거 작업 기록을 현재 코드 기준 용어로 정리한 것이다. 현재 제품 기준은 상위 문서인 [제품 개요](../product-overview.md), [시스템 맵](../system-map.md), [데이터 모델](../data-model.md)을 따른다.

## 정리된 내용

- FlowNote를 문서관리와 현장 지식 축적을 함께 다루는 사내 서버형 제품으로 정리했다.
- 초기 범위는 문서 파일, 문서 버전, 변경 사유, 로그인 권한, 현장 단말기 뷰어, FieldComment, 작업순서, 보고서 보조에 둔다.
- AI 검색과 작업 조언은 현재 구현 범위가 아니라 축적된 데이터를 활용하는 후속 계층으로 둔다.
- 클라이언트는 Windows WPF 설치형 앱을 기준으로 하고, 브라우저 직접 접근은 기본 운영 방식으로 두지 않는다.

## 현재 코드와의 관계

- Windows 앱은 로컬 SQLite를 사용해 문서, 버전, FieldComment, 작업순서, 알림, 보고서, 사용자 계정을 관리한다.
- FastAPI 서버는 SQLite와 서버 로컬 파일 저장소를 기준으로 문서, FieldComment, 작업순서, 보고서 API를 제공한다.
- 문서 구조는 고객이 정하는 폴더 체계를 따른다. BOM 문서 구조는 예시이며 기본 강제 구조가 아니다.
