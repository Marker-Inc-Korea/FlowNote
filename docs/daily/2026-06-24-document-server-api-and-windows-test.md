# 2026-06-24 문서 서버 API와 Windows 테스트 기록

이 문서는 문서 서버 API와 Windows 앱 검증 흐름을 현재 코드 기준으로 정리한 기록이다.

## 서버 API

- 문서 등록, 목록, 상세, 공개 문서 조회, 버전 목록, 새 버전 등록 API가 구현되어 있다.
- 문서 상태와 버전 상태는 별도로 관리한다.
- 태그는 사전형 태그와 문서 연결 태그로 관리한다.
- 열람 로그는 문서별로 등록하고 관리자 권한에서 조회한다.

## Windows 앱 검증 대상

- 문서 등록 화면에서 문서 메타데이터와 파일을 등록한다.
- 문서 목록과 상세 화면에서 버전, 태그, 상태, 열람 이력을 확인한다.
- 관리자 권한 사용자는 파일 감시 후보와 문서 등록 흐름을 사용할 수 있다.
- 현장 사용자는 권한에 따라 다운로드가 제한되고, 문서 열람 이력이 남는다.

## 테스트 기록 보존

스모크 테스트로 생성된 SQLite 데이터, 로그, 등록 문서, 출력 파일은 삭제하지 않는다. Git에는 SQLite를 제외한 테스트 산출물과 로컬 파일 저장소를 올리지 않는다.
