# FlowNote 구현 로드맵

## 완료된 기반

- Windows WPF 앱 골격
- WPF 로컬 SQLite 초기화와 기본 시드
- 로그인, 사용자 관리, role 정책
- 문서 폴더/문서/버전/태그
- 문서 미리보기와 접근 로그
- FieldComment와 첨부
- 알림과 활동 이력
- 작업순서 보드/항목/TV 화면
- 보고서 초안 저장
- 관리자 파일 감시 후보
- FastAPI SQLite 서버와 `/api/v1` API
- 서버 인증 세션과 refresh/logout
- WPF 서버 전송 큐와 서버 ID 매핑
- 테스트 DB와 산출물 보존 규칙에 맞춘 검증 자동화

## 다음 우선순위

4. 보고서 서버 API와 WPF 보고서 흐름 연결
5. 실제 현장 파일/문서 유형별 미리보기 안정화
