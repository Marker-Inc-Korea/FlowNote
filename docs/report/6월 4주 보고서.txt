남상진 이번 주 내용 및 성과

1. 기존 현장 테스트에 일부 기능이 포함되어 들어간 Z시리즈에서 FlowNote를 분리하여서 FlowNote 만 가동하도록
새롭게 제작을을 하는 방향으로 전환

7월에에 테스트베드에서 지금 있는 기능(인수인계, 문서등록, 작업순서) 등에 대해서 실제 도입에 대해서 Z시리즈와의 
통합은 처음부터 잘못된 방향이여서 FlowNote 기능을 분리하도록 하였습니다. 

현장에도 새롭게 만들어진 FlowNote를 사용 요청부터 현장의 교육까지 새롭게 시작하는 방향으로 전환을 준비해야 될것으로
처리 해야 될듯 합니다.

테스트가 새롭게 시작되면서 일단 기존에 넣어준 기능의 제거 작업으로 인해서 테스트 현장의 서버 사용이 불가능 형태로
되어서 별도 서버를 구축해서 처리 해야 할듯합니다.

테스트가 되는 현장의 문서를 제공에 대해서 불가 하다는 현장 담당자의 이야기로 인해서 새롭게 문서를 구하고 기존에
관련된 문서를 새롭게 구해서 프로그램 테스트를 진행 해야 될것으로 판단됩니다.

전체적으로 3주간 일부 기능을 구현해서 테스트 현장에 설치와 테스트가 일부 진행 되었지만 기능이 새롭게 
달라지고 새롭게 설치 해야 되는 제약 사항이 생기면서 현장의 교육부터 새롭게 진행을 해야 되지만, 
추가 설치와 앱의 설치 지원까지 모두 해결하기 위해서는 7월에 테스트 현장의 자산을 사용할 수 있을지는
FlowNote를 보고 결정을 하는 결과가 되었습니다. 

2. Z시리즈에 일부 개발된 기능을 다시 FlowNote로 다시 만들기 시작 하면서 새로운 방법 새로운 UI로 제작을 진행
이후에 현장에 사용가능한 네이티브 방향으로 다시 방향을 잡고 전환 개발 시작 

기존에 Web + 네이티브 하이브리드 방식에서 순수 네이비트 방식으로 변환을 해서 처리 가능하도록 개발을 방향을 
새롭게 정의 하였습니다 .

3. 깃허브에 Z시리즈 Client가 올라간 부분에 대해서 정리 
기존에 Private 저장소를 Public로 변환해서 처리 


남상진 다음주 개인 연구 계획

1. 기존 Z시리즈에 대한 테스트을 위해 제공된 FLowNote 일부 기능이 첨부된 프로그램을 제거 하는 작업 진행
2. 현장 방문 일정에 대해서 7월 전면 취소 및 다시 미팅 및 현장 방문 일정 조율 협의 진행 
3. FlowNote 기능 개발에 더 많은 시간 할애로 완성도 올리기 작업을 진행
