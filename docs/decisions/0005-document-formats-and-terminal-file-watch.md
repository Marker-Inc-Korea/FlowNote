# 0005. 문서 형식과 단말기 파일 감시 역할 분리

## 상태

Accepted

## 결정

FlowNote에서 문서는 HWP, Word, PowerPoint, Excel, PDF, DWG 등 생산 현장에서 사용하는 모든 업무 문서를 포함한다.

단말기 모드는 다음과 같이 분리한다.

- `viewer`: 현장 사용자 단말기
- `admin_support`: 관리자 단말기

`viewer` 모드는 현장 공개 문서 열람, 알림 확인, 현장 코멘트 등록만 수행한다. 로컬 파일이나 폴더를 감시하지 않는다.

`admin_support` 모드는 지정 파일 또는 폴더의 변경을 감지한다. 변경 시간, 크기, 해시값을 비교하고 변경 후보를 만든다. 실제 새 버전 업로드는 관리자가 변경 사유와 버전명을 입력한 뒤 확정한다.

## 근거

현장 사용자 단말기는 단순하고 안전해야 한다. 파일 감시와 업로드 기능은 승인되지 않은 문서 변경 위험을 만들 수 있으므로 관리자급 기능으로 제한한다.

자동 업로드가 아니라 업로드 보조로 설계하면 잘못된 문서 개정을 줄일 수 있다.

## 결과

- `FileObject.file_family`를 둔다.
- `TerminalDevice.device_mode`를 둔다.
- `WatchedFile`과 `FileChangeCandidate`를 둔다.
- 관리자용 파일 감시 API를 제공한다.
- 현장 사용자 단말기는 파일 감시 API를 사용할 수 없다.
