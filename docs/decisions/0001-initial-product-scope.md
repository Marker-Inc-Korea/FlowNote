# 0001. 초기 제품 범위와 도메인 분리

## 상태

Accepted

## 결정

FlowNote는 생산공장 문서 관리 서버를 중심으로 다음 도메인을 분리한다.

- 문서 관리: `Document`, `DocumentVersion`, `FileObject`
- 문서 보조 정보: 태그, 연결 대상, 권한, 이력, 접근 로그
- 현장 단말기: 현장 공개 문서 열람, 알림, 현장 코멘트 등록
- 관리자 지원: 지정 파일 변경 감지와 새 버전 업로드 보조
- 현장 코멘트: 작업 평가, 문제점, 개선 의견, 정형 문구
- 작업내역: 작업지시 문서와 관련 작업 이력
- AI 기능: 검색 인덱스, 문서 검색, 작업 조언, 위험 요소 미리보기

## 근거

문서 파일, 현장 코멘트, 작업내역, AI 검색 결과는 생명주기와 권한 기준이 다르다. 하나의 모델로 합치면 확장과 감사 추적이 어려워진다.

## 결과

- 문서와 파일은 분리한다.
- 문서와 현장 코멘트는 연결하되 같은 모델로 저장하지 않는다.
- 작업내역은 문서와 연결하되 별도 버전 모델을 가진다.
- AI 검색은 원본 데이터를 대체하지 않고 인덱스와 참조를 사용한다.
- 도메인 간 상관관계는 `docs/system-map.md`를 기준으로 유지한다.
